Sample HDF5 Programs and Files

  • For a list of sample programs and corresponding HDF5 files of creating atomic and complex datatypes, see:
    http://hdf.ncsa.uiuc.edu/training/other-ex5/sample-programs/
  • For a list of NASA HDF and HDF-EOS files that were converted to HDF5, see:
    http://hdf.ncsa.uiuc.edu/training/other-ex5/sample-programs/convert/Conversion.html
  • Some HDF5 files can also be found from the NCSA ftp server at:
    ftp://ftp.ncsa.uiuc.edu/HDF/files/hdf5/
  • HDF5 files converted from DOQ, DEM, GEOTIFF, and ESRI Shapefiles can be found at:
    ftp://ftp.ncsa.uiuc.edu/HDF/files/hdf5/geospatial/
