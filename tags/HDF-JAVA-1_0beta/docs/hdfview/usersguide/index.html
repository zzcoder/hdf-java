HDFView User's Guide
Version 1.0, March 2002

The HDFView is a graphic utility designed for viewing and editing the contents of HDF4 and HDF5 files. This document provides the following information:
  • User instructions for HDFView Version 1.0
  • A brief discussion of the HDF object model.
Details of the HDF object model are available from The NCSA HDF Object Model at http://hdf.ncsa.uiuc.edu/hdf-java-html/hdf-object/.
