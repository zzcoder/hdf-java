HDFView User's Guide
Version 1.1, June 2002

The HDFView is a graphic utility designed for viewing and editing the contents of HDF4 and HDF5 files. This document provides instructions of how to use the HDFView. You will find the following information in this document:
  • User instructions for HDFView
  • A brief discussion of the HDF object model. Details of the HDF object model are available from http://hdf.ncsa.uiuc.edu/hdf-java-html/hdf-object/
