The HDFView is a visual tool for browsing and editing NCSA HDF4 and HDF5 files. Using HDFView, you can
  • view a file hierarchy in a tree structure
  • create new file, add or delete groups and datasets
  • view and modify the content of a dataset
  • add, delete and modify attributes
  • replace I/O and GUI components such as table view, image view and metadata view
For information on how to use the HDFViewplease read
the User's Guide

To know how to implement HDFView modules, please read
the Module Guide

Install/Run Instructions

  • Unix (not including Mac OS X):

    • To install, type:   sh ./hdfview_install_{linux, solaris, irix}_{vm, novm}.bin
    • To run, type:   ./hdfview
    • Important Notes:

      OSF1:   Before installing and running HDFView on OSF1, make sure that the Java VM works [try "java -version"]. If you have a problem starting the Java VM, you may have to set the Classic Java Virtual Machine Environment Variable, as follows:   setenv JAVA_CLASSIC_VM "yes"

  • Mac OS X:

    • To install, open hdfview_install_macosx_novm.zip
    • To run, {where_you_install_hdfview}/hdfview.sh
      or click on the HDFView Icon on the desktop

  • Windows:

    • To install, type: hdfview_install_windows_vm.exe
    • To run, select HDFView from the menu, Start -> programs -> NCSA HDFView 2.2 -> HDFView 2.2
