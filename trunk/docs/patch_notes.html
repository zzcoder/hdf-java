hdf-java 2.5 patch 02

Patch 02 was released on May 20, 2009. Patch 02 included the following fixes:

  • All patch 01 fixes. For details, see Patch 01 README.
  • Copying/pasting text data for compound dataset does not work right.
  • TableView does not use (0, 1, 2) order for 3D dataset.
It also added the following capabilities (prototype):
  • Show data pointed by region reference.
  • Add options to display integer data in decimal, hexadecimal, or binary.
  • Use bitmask to show bit(s) of byte data.

For details, see release notes at http://www.hdfgroup.org/ftp/HDF5/hdf-java/v25p2/README.

To obtain the patch, go to http://www.hdfgroup.org/ftp/HDF5/hdf-java/v25p2
