Product Description

The HDF Java Products are revised and reorganized from the earlier Java HDF products. It replaces the old HDF viewer (JHV) and the HDF5 viewer (H5View).

JHV is no longer supported.
H5View is no longer supported.

Product Description
Java HDF Interface (JHI) The Java Native Interface that wraps the HDF4 library. The Java HDF Interface (JHI) is used to access the HDF4 library only
Java HDF5 Interface (JHI5) The Java Native Interface that wraps the HDF5 library. The Java HDF5 Interface (JHI5) is used to access the HDF5 library only
hdf-object package A Java package that implements HDF4 and HDF5 data objects in an object-oriented form. The HDF Java Object Package provides a common standard Java API to access both HDF4 and HDF5 files.
HDFView A visual tool for browsing and editing HDF4 and HDF5 files.

Binary release

The binary release includes jar files for the view package, for the HDF Object Package, the JHI (HDF4 library interface) and JHI5 (HDF5 library interface). The native libraries are available for selected platforms. You can download the packed tar file from:

    ftp://ftp.ncsa.uiuc.edu/HDF/HDF5/hdf-java/bin/{linux,macosx,solaris,windows}

Or the individual file from:
    ftp://ftp.ncsa.uiuc.edu/HDF/HDF5/hdf-java/bin/{linux,macosx,solaris,windows}/hdf-java

Binary file Description
lib/jhdf.jar The Java HDF Interface, accesses the native HDF4 library
lib/jhdf5.jar The Java HDF5 Interface, accesses the native HDF5 library
lib/jhdfobj.jar The common HDF object package, ncsa.hdf.object
lib/jhdf4obj.jar The HDF4 object package, ncsa.hdf.object.h4
lib/jhdf5obj.jar The HDF5 object package, ncsa.hdf.object.h5
lib/jhdfview.jar The ncsa.hdf.view package, for HDFView only.

Source

The source distribution includes all of the Java source to build the jar files, plus the C source for the native methods of the JHI and JHI5.

The JHI requires the HDF4.2r1 library (binary) and the JHI5 requires the HDF5 1.6.5 library (binary). The HDF libraries are not included in the Java source. They can be obtained from: http://hdf.ncsa.uiuc.edu/
