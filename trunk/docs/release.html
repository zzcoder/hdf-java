hdf-java 2.6 release (January 15, 2010)

Version 2.6 of the HDF Java Products was built with HDF4.2r5 and HDF5 1.8.4 with the 1.6 API Compatibility Macros, using jdk 1.6.x.

To obtain hdf-java 2.6 (HDFView installers, Pre-built binaries, and Source code), go to http://www.hdfgroup.org/ftp/HDF5/hdf-java/.
For more information on the current release, please read About This Release.
