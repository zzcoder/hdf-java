HDF-Java 2.99 release (trunk)

HDF-Java 2.99 was built with HDF 4.2.x and HDF5 1.8.xx using jdk 1.7.x. This is a maintenance release with no major changes in functionality. This release includes bug fixes and some minor enhancements.
