HDF-Java 2.99 release (under development)

HDF-Java 2.99 was built with HDF 4.2.x and HDF5 1.8.x using jdk 1.7.x. This is a maintenance release with no major changes in functionality. This release includes bug fixes and some minor enhancements. The install programs now use native platform installs:

  • Windows, HDFView-2.99.0-winXX.exe (where XX is 32 or 64)
  • Linux, HDFView-2.99.0-Linux.sh
  • Mac OS X, HDFView-2.99.0.dmg
