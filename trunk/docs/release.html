hdf-java 2.7 release (November, 2010)

This release was built with HDF 4.2.5 and HDF5 1.8.6 using jdk 1.6.x.

Major changes in HDF5 1.8.6 includes the following:
  • 101 methods were added to JHI5 to support HDF5 1.8
  • New methods were added to the HDF-Java Object Layer to support HDF5 1.8 features
  • Some HDF5 1.8 features such as external links were added to HDFView
  • This release includes over 20 major bug fixes and enhancements

For more information on the current release, please read About This Release.
