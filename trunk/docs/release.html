hdf-java 2.7 release (November, 2010)

This release was built with HDF 4.2.5 and HDF5 1.8.6 using jdk 1.6.x.

HDF5 1.8.6 Added new API's for improved performance. For more information,

No major change in hdf-java since hdf-java 2.6. Some minor changes include:
  • Support of HDF5 1.8 in JNI, HDFJava Object Package and HDFView
  • Add features in HDFView to move objects within the same HDF5 file
  • Fixed the problem to copy large datasets
  • Add functionality to import/export data to/from a binary file.


For more information on the current release, please read About This Release.
