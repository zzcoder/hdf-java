HDF-Java 2.9 release (November 15, 2012)

This release is the last one to support 32-bit Mac and 32-bit Solaris. Future releases will not support these two platforms anymore.

HDF-Java 2.9 was built with HDF 4.2.8 and HDF5 1.8.10 using jdk 1.7.x. It is a maintenance release. No Major changes in functionalities. It mainly includes bug fixes and some minor enhancements.

Some highlights include the following:
  • Fixed 29 bugs
  • Added 10 minor enhancements, e.g.
    • add feature to show groups/attributes in creation order
    • exclude fill values in data calculation
    • add 'reload' option to quickly close and reopen a file

For more information on the current release, please read About This Release.
