HDF-Java 2.10 release (November 15, 2013)

HDF-Java 2.10 was built with HDF 4.2.9 and HDF5 1.8.12 using jdk 1.7.x. This is a maintenance release with no major changes in functionality. This release includes bug fixes and some minor enhancements.

Here are the highlights.

  • Fixed xx bugs including the following:
    • Fixed HDFView to display 64-bit unsigned integers
    • Fixed HDFView to display SCALAR arrays
    • Fixed a fill value problem on Mac machines
    • Fixed ImageView showing the wrong values for 3D arrays
  • Added xx minor enhancements including the following:
    • Added a feature to show groups/attributes in creation order
    • Excluded fill values from data calculations
    • Added a 'reload' option to quickly close and reopen a file

For more information on the current release, please read About This Release.
