HDF-Java 2.8 release (December 15, 2011)

This release was built with HDF 4.2.6 and HDF5 1.8.8 using jdk 1.6.x.

Major changes includes the following:
  • More methods were added to HDF5 JNI wrapper
  • Unit tests were added to all major HDF5 JNI functions
  • This release includes over 35 enhancements and bug fixes

For more information on the current release, please read About This Release.
